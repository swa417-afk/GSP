# Governance

## Project Structure

The Glass Substrate Protocol (GSP) is an open standard for governed AI systems
with tamper-evident accountability logging.

### Author

- **Sierra N. Warren** - Protocol author and initial maintainer

The author role is technical, not controlling. This is an open standard that
anyone may implement, extend, or fork.

## Decision Making

### Protocol Changes

Major protocol changes follow this process:

1. **Proposal**: Submit via GitHub issue or pull request
2. **Discussion**: Public technical review period
3. **Implementation**: Reference implementation and testing
4. **Adoption**: Community adoption indicates acceptance

### Repository Maintenance

- Pull requests are reviewed for technical merit
- Maintainers may merge changes that improve the protocol
- Breaking changes require clear justification and migration guidance
